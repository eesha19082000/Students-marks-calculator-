# Student Marks Calculator

## Project Description
This is a simple Python project that calculates a student's total marks, percentage, and grade based on marks entered for 5 subjects.

## Features
- Takes student name as input
- Accepts marks for 5 subjects
- Calculates total marks
- Calculates percentage
- Displays grade based on percentage

## Grade Criteria
- 90 and above = A+
- 80 to 89 = A
- 70 to 79 = B
- 60 to 69 = C
- 50 to 59 = D
- Below 50 = Fail

## Technologies Used
- Python

## How to Run
Run the file using:
python student_marks_calculator.py
